03 · Pre-procesamiento

Preparación de los datos

📂 Carga y transformación

data_bank <- read.csv("Bank_Personal_Loan.csv")
data_bank$Education <- factor(data_bank$Education)
data_bank$Personal.Loan <- factor(data_bank$Personal.Loan)
data_bank$CCAvg <- as.numeric(gsub("/", ".", data_bank$CCAvg))

● Education → factor (3 niveles)

● Personal.Loan → factor (No / Yes)

● Variables binarias → factor

● CCAvg → numeric (1/60 → 1.60)
